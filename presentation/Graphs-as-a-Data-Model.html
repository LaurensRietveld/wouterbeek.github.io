IRIs (Internationalized Resource Identifiers)

Globally unique & unambiguous terms

‘Mew’ the Pokemon
http://dbpedia.org/resource/Mew_(band)
‘Mew’ the band
https://demo.triply.cc/academy/pokemon/id/pokemon/mew
Etc.

Literals

Some things cannot be easily expressed with IRIs.

Natural language terms
‘Mew’; ‘ミュウ’; etc.
Numbers
0.333333; 1,000,000,000; etc.
Date/time points
2018-01-01T09:30:45.0Z
Etc.

Syntax for terms

IRI
<https://demo.triply.cc/academy/pokemon/id/pokemon/mew>
Literal
Language-tagged string
"ミュウ"@ja
Typed literal
"40"^^<http://www.w3.org/2001/XMLSchema#integer>

Abbreviated syntax for IRIs

Prefix declaration

prefix pokemon: <https://demo.triply.cc/academy/pokemon/id/pokemon/>
prefix xsd: <http://www.w3.org/2001/XMLSchema#>

Abbreviated IRIs

pokemon:mew
pokemon:pikachu
"40"^^xsd:integer
