1 IRI strategy

RDF aliases:

  • cbs abbreviates https://cbs.nl/
  • cbs-def abbreviates https://cbs.nl/def/
  • cbs-id abbreviates https://cbs.nl/id/

2 Cleaning

I removed all occurrences of value ‘-99999999’ which is documented to denote values that unknown, untrustworthy or secret.

All occurrences of value ‘-99999999.0’ are removed. This value is not documented but which may be a typo of ‘-99999999’.

All occurrences of the empty string are removed.

Property WATER has very few occurrences of value B which I'm assuming to be mistakes and are therefore removed.

3 Properties

All columns in the tab-separated values files have been described in a vocabulary using RDFS. This vocabulary is used to automatically transform the data into datatypes RDF. There are too many properties to enumerate them all, but they can be found in the RDFS vocabulary graph and many of them can be seen in the example below.

3.1 cbs-def:water

Property WATER is the only property that appears in the data but that is not documented. I'm renaming this to cbs-def:water. Its values are either JA or NEE, converted to xsd:boolean (with the exception of very few occurrences of value B which I assume to be mistakes).

3.2 cbs-def:buurtNaam & cbs-def:gemeenteNaam & cbs-def:wijkNaam

These three properties are tagged as Dutch labels (@nl). The values for property rdfs:label are constructed out of these properties.

3.3 Percentage values

Many properties denote percentages. These are currently transformed to xsd:nonNegativeInteger, but it would be better to transform them to a datatype that denotes percentages.

3.4 Category values

Few properties have a limited number of possible values for which we could create IRIs. These are:

  • cbs-def:dekkingspercentage with values 1 through 6.
  • cbs-def:indelingswijziging with values 1 through 3.
  • cbs-def:stedelijkheid with values 1 through 5.

3.5 Surface values

Few properties denote a surface in ‘hectaren’. For these we should find a surface-denoting datatype:

  • cbs-def:oppervlakteLand
  • cbs-def:oppervlakteWater
